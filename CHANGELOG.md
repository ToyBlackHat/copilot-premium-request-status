# Changelog

All notable changes to this project are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.1] - 2026-02-27

### Changed
- Rounded all aggregated usage values to 2 decimal places to avoid floating-point artifacts from API responses.
- Standardized status bar and tooltip numeric display to `XX.XX` format.
- Updated README VSIX install example to version `0.1.1`.
